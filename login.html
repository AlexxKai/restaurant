Autenticación: Crear páginas de registro e inicio de sesión. Utiliza componentes CSS de Materialize para formularios, botones y tarjetas para mejorar la experiencia del usuario. NO es obligatorio utilizar una base de datos para completar la tarea. 2. 2. Página de Contacto: Diseñe una página web de Contacto con un formulario personalizado. Utilice en ella los siguientes componentes CSS de Materialize: Fichas para etiquetar. Casillas de verificación para seleccionar múltiples opciones. Date Picker para seleccionar fechas. Range para seleccionar un rango de valores. Select para menús desplegables. Conmutadores para opciones binarias. Entradas de Texto para la entrada del usuario. 3. Menú de restaurante: Crear páginas separadas para el primer plato, segundo plato y postres. Utilizar tarjetas CSS de Materialize para mostrar cada elemento del menú. Utiliza insignias para indicar ofertas especiales o etiquetas dietéticas. 4. Página de aterrizaje: Crear una página de aterrizaje con la información general sobre el restaurante. 5. Otros componentes: Utiliza los siguientes componentes CSS de Materialize al menos una vez en todo el sitio web: Insignias: muestra insignias en los elementos del menú para ofertas especiales o categorías. Botones: utiliza botones para acciones como enviar formularios o navegar por el menú. Migas de pan: utilice migas de pan para facilitar la navegación por el menú. Tarjetas: utilice tarjetas para mostrar información detallada sobre cada elemento del menú. Colecciones: utilice colecciones para presentar una lista de elementos (por ejemplo, elementos de menú). Botón de acción flotante: utilice un botón de acción flotante para un enlace rápido al formulario de contacto. Pie de página: cree un pie de página con información relevante sobre el restaurante. Iconos: integre iconos para aumentar el atractivo visual y mejorar la comprensión del usuario. Barra de navegación: diseñe una barra de navegación adaptable para facilitar la navegación. Paginación: implementar la paginación si hay varias páginas para los elementos del menú. 6. Estilo y capacidad de respuesta: Asegúrese de que el sitio web sea visualmente atractivo con un estilo coherente. Haz que el sitio web sea responsivo utilizando el sistema de cuadrícula CSS de Materialize y utilidades responsivas. 7. Pruebas: Prueba el sitio web en diferentes dispositivos y navegadores para garantizar una experiencia de usuario fluida. Nota: Aunque se menciona que una base de datos no es obligatoria, puedes considerar incorporar algunas funcionalidades de JavaScript para una experiencia más dinámica, como validación o interacciones del lado del cliente.
